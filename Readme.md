Follow the below steps to get access to this:

1. Download [Obsidian](https://obsidian.md/download)
2. Install [Relay Plugin](https://relay.md/relay)
3. Obsidian → Settings → Relay → Join Relay Server → "dunked-beryl-mature" → Join
4. Download [_Images](https://github.com/hansaskov/Dungeon-and-Dragons/releases/download/1.0.0/_images.zip) an place them adjecent to the Campaign folder
